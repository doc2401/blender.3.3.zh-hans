线条画修改器

The Line Art modifier generates stylized line art from the scene or selected source collection or objects.

Note

由于当前缺少全局缓存，每个线条画修改器将自行运行整个遮挡计算。因此，如果您有多个线条艺术修改器来选择场景的不同部分（以应用不同的样式等），则评估将花费更长的时间。有计划将来纠正这一点，但这是目前已知的局限性。

选项

使用硬盘缓存

通过使用堆栈中第一个线条画修改器的缓存场景数据来优化渲染。此选项有某些设置不可用的缺点。

This option only available when you have more than one Line Art modifier is the same modifier stack and the modifier is not the first Line Art modifier in the stack.

源类型

应从哪种类型的几何源生成线条艺术。

场景、集合、物体

物体/集合

根据源类型，可以选择集合或物体作为源几何体。

反转集合过滤

从指定集合中选择除行之外的所有内容。

Note

除非在物体或集合中指定其他方式，否则线条画仍将加载并计算整个可见场景以产生正确的遮挡结果。线条艺术 用法 属性。

层

The Grease Pencil 层 to put the result in.

材质

蜡笔 材质 以生成笔画。

线条宽度

线条图生成的笔画给出了此厚度。

不透明度

线条画生成的笔画会给出此不透明度。

边类型

Line Art可以识别不同的边缘类型。所选的边缘类型将包含在结果中。

过滤

选择来自光照或阴影区域的特征线。不会影响投影阴影和光线轮廓，因为它们位于边框处。

无

不根据照明区域过滤任何线条。

照亮

仅从被照亮区域选择线条。

着色

仅从阴影区域中选择线条。

照亮（封闭形状）

仅从被照亮区域选择线条，并将轮廓、光线轮廓和阴影线组合到封闭形状内。

创建标签 Create Tabs

轮廓

从轮廓线生成笔画。当边缘变成正面/背面的分离线时。也可以通过单击反转按钮来反转轮廓。

无

不根据照明区域过滤任何线条。

照亮

仅从被照亮区域选择线条。

着色

仅从阴影区域中选择线条。

照亮（封闭形状）

仅从被照亮区域选择线条，并将轮廓、光线轮廓和阴影线组合到封闭形状内。

折痕

在边缘角度足够小的地方生成笔画。

折痕阈值

Angles smaller than this will be treated as creases. Crease angle priority: object line art crease override > mesh auto smooth angle > line art default crease.

交集

Generate stokes where lines intersect between faces.

材质边界

生成边缘将具有不同材质的面分开的描边。

边标记

从Freestyle边标记生成描边。

松散

为不形成 面 的边生成笔画。

灯光轮廓

Generate light/shadow separation lines from a reference Light Object.

投射阴影

使用光源物体投影轮廓线。

选项

Allow Overlap

允许边缘具有多个重叠类型。这将为每个重叠类型创建单独的笔画。

参考

灯光物体

Use this light object to generate :ref:Light Contour <bpy.types.LineartGpencilModifier.use_light_contour>.

阴影摄像机尺寸

This value represent the "Orthographic Scale" of an ortho camera. If the camera is put at the lamps position with this scale, it will represent the coverage of the shadow "camera".

近端

阴影摄像机的近裁距离。

远端

阴影摄像机的远裁距离

几何处理

自定义

Use custom camera instead of the active camera for calculating strokes. Useful when baking multiple shots in different angle as well as for motion graphics effects.

重叠边作为轮廓

此选项允许将重叠边缘（例如来自边缘拆分修改器或导入的几何体，其中两个边缘占据完全相同的空间）绘制为轮廓。启用此选项将稍微减慢计算速度，但它将处理边缘重叠的情况，而不会产生错误的遮挡结果。

实例物体

此选项使粒子和其他实例化对象能够加载用于线条艺术计算。场景中存在大量实例化对象时，将对性能产生影响。

截断区间

启用后，线条艺术将在近或远裁剪平面切割模型的位置以轮廓类型生成裁剪线。否则不会有线条。

平滑上的折痕

允许在光滑的表面内部显示折痕边。

锐边上的折痕

允许在尖锐的边缘出现折痕。

强制背面剔除

移除所有背面以加快计算速度，这将创建与禁用时不同的遮挡级别的边缘。

遮挡

范围

如果启用，修改器将选择在起始值和结束值之间具有遮挡级别的线。

水平

希望遮挡级别被选择为线条画结果。值为0表示可见线（无遮挡）。值为1意味着选择正好被一层面遮挡的线。

材质遮罩

如果启用，Line Art将仅选择材质已设置特定遮挡掩码的某些面遮挡的线条。

遮罩组

要选择已被选定的 材质遮罩 遮挡的边缘。

完全匹配

如果启用，将仅选择使用精确遮罩位组合遮挡的线。否则，将选择被任何一个指定的材质遮罩遮挡的线条。

交叉

允许您选择两个集合之间相交的边缘。

集合

从 线条画集合 属性中匹配的遮罩位。

完全匹配

需要匹配所有交叉遮罩，而非仅仅一个。

面标记过滤

面标记过滤 可用于通过使用Freestyle面标记手动控制哪些功能边缘产生笔画。

反转

反转面标记过滤。

边界

基于面标记边界过滤特征线。

保持轮廓

过滤时保留等高线。

链形

链
与轮廓的交集

允许将交叉线与轮廓线链接在一起。

Note

启用此选项将导致交叉边缘类型不明确。无法与附近的任何轮廓线连锁的交叉线将保持为交叉线。

全部线条

启用此选项将导致所有线条具有相同类型的轮廓并被链接在一起。

松散边

允许将不构成面的浮动边链接在一起。

松散边作为轮廓

不形成面的边将被归类为轮廓线。

保留细节

不要在每次遮挡变化时分开，保持最初链条上的小细节。当不保存细节时，将产生更顺畅的结果。

几何空间

对链式而非图像空间使用几何距离。

图像阈值

如果二维图像间距在指定的阈值内，则允许将短段的端点链接在一起。

平滑容差

应用于锯齿状链条的平滑强度。

拐角拆分

以此角度指定的尖锐 "转弯" 点分割链条。

顶点权重传递

过滤源

如果源网格具有名称以此文本开头的顶点组，则顶点权重信息将传输到蜡笔笔画中的权重组中。

匹配输出

将过滤的对象顶点权重传输到蜡笔权重组中，其名称与过滤的对象顶点权重组名称相同。

目标

如果 匹配输出 关闭，则必须指定目标顶点组。如果有多个权重组复制到目标中，则将最大权重值复制到目标中。

合成

过扫描

为了优化渲染，Blender仅渲染处于相机视图中的对象边缘的笔触。然而，这种优化可能会导致笔画在图像的边缘处突然结束。

此值通过在摄像机视图外添加边距以继续计算笔画来防止此错误。

图像边界修剪

修剪图像边界上的所有边线（包括过扫描区域）。

深度偏移

稍微将笔画朝向摄像机移动，以避免剪裁，同时保留视口的深度。此选项不可用，除非 显示在前面 被禁用。

朝向自定义摄像机

将笔画偏移到所选相机（参见上面的 自定义相机 ）而不是活动相机。

烘焙

烘焙线条画

在场景中的 开始、* 结束* 帧范围内为活动蜡笔对象烘焙线条画笔画。烘焙线条画（全部）烘焙包含至少一个线条画修改器的所有蜡笔对象。烘焙后，烘焙过的线条画修改器将自动停用。

清除已烘焙线条画

清除活动蜡笔对象的场景帧范围内的烘焙线条画帧。清除烘焙线条画（全部）对所有至少包含一个线条画修改器的蜡笔对象应用相同的操作。

Warning

如果您手动绘制了线条画运行位置的帧范围中的任何内容，此操作也将清除这些笔画！

持续

在不清除烘焙笔画的情况下重新激活特定的线条画修改器。这对于分开处理框架的多个部分非常有用。
