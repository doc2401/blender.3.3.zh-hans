  • 笔刷设置

笔刷设置

绘制需要笔刷，而在Blender 权重绘画模式 下的工具栏中有一个笔刷面板。

笔刷

在 数据块菜单 中可以找到预定义的笔刷预设。还可以根据需要创建自己的自定义预设。

半径

此选项控制笔刷的半径，以像素为单位。F 用于通过拖动鼠标交替改变笔刷大小，然后 LMB (笔刷的纹理应该在圆圈内可见)。然后在使用 F 时键入一个数字并回车，可以数值控制笔刷大小。

压力

Brush size can be affected by enabling the pressure sensitivity icon, if you are using a Graphics Tablet.

强度

笔刷绘制时有多大力度。

压力

Strength can be affected by enabling the pressure sensitivity icon, if you are using a Graphics Tablet.

模式
笔画

只在笔画上作画。

填充

只在填充区上涂抹。

描写和填充

在笔画上涂抹，填充区域。

游标

游标 设置见全局笔刷设置。

拾色器

笔刷的颜色，见 拾色器。

Note

请注意，在 sRGB 工作 空间 中的顶点绘制，相同颜色的 RGB 表示形式在绘制工具和线性空间中的材质之间会有所不同。

调色板

The active Color Palette. See 调色板.

衰减

衰减 设置见全局笔刷设置。
