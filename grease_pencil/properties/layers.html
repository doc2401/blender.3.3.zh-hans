图层列表

蜡笔对象各自具有用于在 列表视图 中分组和排列笔画的2D图层列表。任何笔画只能属于单个2D图层。列表中始终只有一个活动图层（所选图层）。绘制时，新笔触将添加到活动图层。默认情况下，视口中图层的视图顺序是从上到下。

Every layer correspond to a channel in the Dope Sheet editor (in Grease Pencil mode). See Dope Sheet for more information.

图层也可以与修改器一起使用，仅影响部分绘图。查看 修改器 来获得更多信息。

Tip

有时，您没有处理的图层可能会分散注意力。激活叠加中的 淡入非活动图层 以控制非活动图层的不透明度。有关详细信息，请参阅 叠加 。

图层名称旁边有四个图标按钮，用于控制图层的常见属性：

遮罩（遮罩图标）

Toggle the affect of Masks on the layer.

洋葱皮（洋葱皮图标）

切换 洋葱皮 时，是否使用该图层。

隐藏（眼睛图标）

在视图和渲染中切换图层的可见性。

锁定（锁上图标）

切换图层不可编辑。

图层专用项

用于处理图层的操作符。

Duplicate Layer

制作所选图层的精确副本，附加数字以区分其名称。

复制空白关键帧

生成选定图层的副本，但关键帧为空。用于轻松预设空关键帧以用于清理或填充过程。

显示全部

打开列表中每个图层的可见性。

隐藏其他

关闭列表中除活动层之外的每个层的可见性。

全部锁定

锁定列表中所有图层的编辑性。

全部解锁

解锁列表中所有图层的编辑性。

淡化非活动几何体

自动锁定列表中除活动图层外的每个图层的版本。这样，您可以避免在其他图层中进行不必要的更改，而无需每次都锁定它们。

Disallow Locked Materials Editing

避免在图层中编辑锁定的材质。禁用后，即使它们在材质列表中被锁定，也可以编辑任何材质。

Merge Down Shift-Ctrl-M

Combine the selected layer with the layer below, the new layer keeps the name of the lower layer.

Merge All

Combine all layers into the active layer.

从图层复制图像

将现用图层复制到选定的蜡笔对象。

全部复制到所选物体

将所有图层复制到选定的蜡笔对象。

Visibility (screen icon)

Toggle whether the active layer is the only one that can be edited and is visible.

Isolate Lock (padlock icon)

Toggle whether the active layer is the only one that can be edited.

Below the layers list there are additional common settings:

混合

要执行混合的操作的图层。见 颜色混合模式。

不透明度

用于设置图层的不透明度。

Use Lights

启用后，图层会受到光源的影响。

遮罩组

In a List view of layers affected by a layer mask. See Masks for more information.

变换

允许每层位置、旋转和缩放变换。

Adjustments

Tint Color

Color that tint any material colors used in the layer.

系数

Controls the amount of tint color to apply.

Stroke Thickness

Thickness value that override the strokes thickness in the layer.

关系

Parent/Type

Select a Parent object and Type to manipulate the layer. The layer will inherit the transformations of the parent, this is especially useful when rigging for cut-out animation.

通道编号

层编号值可与某些修改器一起使用，以限制仅对某些区域的更改。

更多信息见 修改器。

视图层

定义用于 "蜡笔" 层的 "视图层" 。如果为空，则该图层将包含在所有视图图层中。这对于分离绘图部件 合成 非常有用。

Disable Masks in Render

If enabled no masks on the layer are included in the view layer render.

显示

Custom Channel Color

Sets the color to use in the channel region of the Dope Sheet.

Show Only On Keyframed

Makes the layer visible in the viewport only if it has a keyframe in the actual frame. This helps for example when you are in the inking process using the Fill tool and want to only see the strokes that are in the actual frame to avoid fill in unwanted regions.
